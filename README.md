# Auckland Region Newsletter Generator

A browser-based tool for generating Auckland Region Freemasons newsletters with:

- Structured event management
- Outlook-compatible HTML output
- RSVP and refectory tracking
- Highlighted events (Featured / Official Visit / Important)

## How to Use

1. Open the tool:
   https://yourusername.github.io/auckland-newsletter-generator/

2. Fill in the sections:
   - Header
   - Body
   - Events
   - Footer

3. Use the buttons:
   - Copy HTML → paste into Outlook Web
   - Export EML → open in Outlook
   - Download HTML → backup/archive

## Notes

- Designed for Outlook compatibility
- Images are embedded at original quality
- Events support RSVP and attire tracking
